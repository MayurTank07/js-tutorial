
    WHAT IS AN ARRAY ? 

    -> ARRAY IS A COLLECTION OF DATA WHICH IS STORED IN A SINGLE VARIABLE 

    -> ARRAY IS REPRESENTED BY [ ] 

    -> ARRAY HAS INDEX NO WHICH STARTS FROM 0

    -> DATA INSIDE OF AN ARRAY IS KNOWN AS ELEMENTS
