
- WHAT IS DATATYPES ? 

--> DATATYPES IS USED TO SPECIFY WHICH TYPE OF DATA WE ARE GOING TO STORE INSIDE OF VARIABLE.

e.g -- 
    var name = "dexter";  --> string data

    var age = 24;  --> number data

    var marks = 89.89;  -->  float data (float means which has a decimal point)

        d. null
        e. boolean

    2. NON PRIMITIVE DATATYPE
        a. object
